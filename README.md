# Restaurant Management System in C

This project is a console-based Restaurant Management System developed using the C programming language. The program allows customers to order food items from different categories such as Starters, Main Course, Desserts, and Beverages.

## Features

* Interactive menu-driven interface
* Multiple food categories
* Order selection and quantity input
* Automatic bill calculation
* Display of ordered items
* Final bill generation
* User-friendly navigation using loops and conditional statements

## Technologies Used

* C Programming Language
* GCC Compiler
* Linux/Windows Terminal

## Learning Outcomes

This project helped in understanding:

* Variables and Data Types
* Conditional Statements
* Loops (do-while)
* Functions
* Arrays and Strings
* File Management using Git and GitHub
